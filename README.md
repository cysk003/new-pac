## # Project Overview
Welcome to the **Network Routing & Congestion Control Laboratory**. This repository is a long-term open-source research initiative (active since 2015) dedicated to the study and optimization of modern network protocols, routing logic, and kernel-level congestion control algorithms.

Our primary goal is to provide developers and researchers with a modular framework to analyze network performance, minimize latency, and improve throughput across heterogeneous network environments.

## 🔬 Core Research Modules
* **Routing Table Optimization (PAC)**: Implementing advanced Proxy Auto-Config logic to demonstrate efficient traffic steering and domain-based routing.
* **Kernel Congestion Control**: Shell-based automation for deploying and benchmarking BBR (Bottleneck Bandwidth and Round-trip propagation time) variants.
* **Protocol Stack Analysis**: Experimental implementation of various tunneling protocols (Sing-Box, Shadowsocks, etc.) to evaluate their overhead and security characteristics.
* **Legacy Compatibility Layer**: Maintaining support for historical protocols to ensure backward compatibility during research transitions.

## 🛠 Project Structure
* `/scripts`: A collection of automated deployment utilities for server-side optimization.
* `/example`: Reference configurations for standardized network research environments.
* `bbr.sh`: Kernel-level congestion control activation and tuning.
* `sing-box-plus.sh`: Integrated framework for deploying high-performance network cores.

## 📈 Community Impact
Since its inception, this project has served as a vital resource for the developer community, facilitating thousands of experimental setups and providing a foundation for network performance studies worldwide.
